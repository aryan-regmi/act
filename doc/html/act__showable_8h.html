◆ ACT_SHOWABLE_NEW

#define ACT_SHOWABLE_NEW (   T,
  allocator,
  show_func,
  error_code 
)     act_showableNew(allocator, sizeof(T), (AsStringFnPtr)(show_func), error_code)

Create a new act_Showable type for a struct of type T.

This is a convinience macro to avoid casts and sizeof when calling act_showableNew.

Parameters
[in]TThe type of struct to create the act_Showable for.
[in]allocatorThe allocator used to make internal memory allocations.
[in]show_funcThe function used to convert a value of a given type to a string.
[out]error_codeThe error code (act_ShowableError) of the operation.
Returns
A pointer to an allocated struct of struct_size.
Note
This function allocates 16 + sizeof(T) bytes of memory (The 16 bytes is the size of the internal header used to store showable information).
See also
act_showableFree
